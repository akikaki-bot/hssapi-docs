## GET /v1/school/\{id\}

:::warning
認証が必須です。
:::

学校情報を取得します。

### リクエスト

```http
GET /v1/school/:id
```

### レスポンス ( Example )

```js

    "details" : {

                                "istooBig" : boolean,

                        "homework" : [
```

|  値名  |  型  | 説明  |
| ---- | ---- | ---- |
|  schoolId  |  Snowflake (BigInt or String)  | 学校のuniqueIDです。  |
|  details  |  Object  | 学校の詳細情報です。  |
|  details.type  |  enum `SchoolType`  | 学校のタイプです。  |
|  details.name  |  String  | 学校名です。  |
|  details.id  |  Snowflake (BigInt or String)  | 学校のuniqueIDです。  |
|  details.ownerId  |  Snowflake (BigInt or String)  | 代表者（作成者）のユーザーIDです。  |
|  details.admins  |  Array\<string\>  | 共同管理者のユーザーIDです。配列を返します。  |
|  userDatas  |  Array\<UserDatas\>  | ユーザーのデータです。  |
|  userDatas.grade  |  Number  | 学年です。  |
|  userDatas.class  |  Number  | クラスです。  |
|  userDatas.timelineData  |  Object  | 一週間の時間割の時間割が格納されます。  |
|  userDatas.defaultTimelineData  |  Object  | 基本時間割が格納されます。  |
|  userDatas.eventData  |  Object  | イベントデータが格納されます。  |
|  userDatas.defaultTimelineIndex  |  Number  | 基本の教科数です。  |
|  userDatas.timelineData.[date]  |  Array\<Object\>  | [date]の時間割です。  |
|  userDatas.defaultTimelineData.[date]  |  Array\<Object\>  | [date]曜日の時間割です。  |
|  userDatas.eventData.[date]  |  Array\<Object\>  | [date]曜日のイベントです。  |
|  userDatas.timelineData.[date].name  |  String  | 教科名です。  |
|  userDatas.timelineData.[date].place  |  String  | 場所です。  |
|  userDatas.timelineData.[date].homework  |  Array\<Object\>  | 宿題データです。  |
|  userDatas.timelineData.[date].homework.name  |  String  | 宿題名です。  |
|  userDatas.timelineData.[date].homework.istooBig  |  Boolean  | とてもやるのに時間がかかるものかを判別します。  |
|  userDatas.timelineData.[date].homework.page  |  Object  | ページ数です。  |
|  userDatas.timelineData.[date].homework.page.start  |  String  | 開始ページ数です。  |
|  userDatas.timelineData.[date].homework.page.end  |  String  | 終了ページ数です。  |
|  userDatas.timelineData.[date].homework.page.commnet  |  String  | 何か補足です。  |
|  userDatas.defaultTimelineData.[date].name  |  String  | 教科名です。  |
|  userDatas.defaultTimelineData.[date].place  |  String  | 場所です。  |
|  userDatas.defaultTimelineData.[date].homework  |  Array\<Object\>  | 宿題データです。  |
|  userDatas.defaultTimelineData.[date].homework.name  |  String  | 宿題名です。  |
|  userDatas.defaultTimelineData.[date].homework.istooBig  |  Boolean  | とてもやるのに時間がかかるものかを判別します。  |
|  userDatas.defaultTimelineData.[date].homework.page  |  Object  | ページ数です。  |
|  userDatas.defaultTimelineData.[date].homework.page.start  |  String  | 開始ページ数です。  |
|  userDatas.defaultTimelineData.[date].homework.page.end  |  String  | 終了ページ数です。  |
|  userDatas.defaultTimelineData.[date].homework.page.commnet  |  String  | 何か補足です。  |
|  userDatas.eventData.[date].name  |  String  | イベント名です。  |
|  userDatas.eventData.[date].place  |  String  | 場所です。  |
|  userDatas.eventData.[date].timeData  |  Object  | 時間データです。  |
|  userDatas.eventData.[date].timeData.start  |  String  | 開始時間です。  |
|  userDatas.eventData.[date].timeData.end  |  String  | 終了時間です。  |
|  userDatas.eventData.[date].timeData.isEndofDay  |  Boolean  | 終日続くものかを判別します。  |
