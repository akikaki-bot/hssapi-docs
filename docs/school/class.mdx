# GET /v1/school/:id/class

:::warning
認証が必須です。
:::

登録済みのクラスを返します。

### Response

#### 例：1学年に1から4組が登録されている場合

```json
{
    "1" : [
        1,
        2,
        3,
        4
    ]
}
```

####  例2：1学年に1から3組が、2学年に1から4組が登録されている場合

```json
{
    "1" : [
        1,
        2,
        3
    ],
    "2" : [
        1,
        2,
        3,
        4
    ]
}
```

### Error Responses

- HTTP 401

  - 権限がありません。このエンドポイントを実行するには最低でも管理者権限が必要です。

- HTTP 404

  - 学校に登録されたクラスが存在しません。
